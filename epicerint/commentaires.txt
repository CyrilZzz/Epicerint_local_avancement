Ne pas stocker les images dans la base de données !
Storing images in a database table is not recommended. There are too many disadvantages to this approach.
Storing the image data in the table requires the database server to process and traffic huge amounts
of data that could be better spent on processing it is best suited to. A file server can process such image files much better.
